# 🛒 MultiMart - Modern E-Commerce Frontend

<div align="center">

### 🚀 React.js | Context API | Razorpay | Responsive UI

A modern, scalable, and user-friendly E-Commerce Frontend built with **React.js** that delivers a seamless online shopping experience with authentication, cart management, and secure payment integration.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/narayana-yanamadala/ecommerce-frontend.git
```

### Navigate to Project

```bash
cd ecommerce-frontend
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm start
```
